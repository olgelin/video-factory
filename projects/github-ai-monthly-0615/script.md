六月GitHub AI月榜，这7个项目改变了整个AI开发生态。

第一个，LangChain，13万9千星。不是新项目，但这个月它完成了最关键的转型——从一个LLM工具库升级成了Agent工程平台。现在所有做AI Agent的公司，底层基本都在用它。它就像AI开发的操作系统，你可以不喜欢它，但你绕不开它。

第二个，TradingAgents，8万6千星。这个月增长最猛的项目。它不是让一个AI炒股，而是模拟了一个完整的交易公司：分析师Agent做研究，交易员Agent下单，风控Agent管止损，多个AI各司其职。华尔街看了都要抖三抖。

第三个，Ponytail，6月12号发布，一周8600星，本月最炸裂的新项目。核心理念一句话：让AI像最懒的资深开发一样思考。不是写更多代码，而是判断哪些代码根本不该写。最好的代码，就是不存在的代码。

第四个，claude-mem，8万2千星。Agent开发者的刚需产品。你的Agent每次开新会话就失忆，之前干了什么全忘了。claude-mem给Agent装上永久记忆，跨会话自动恢复上下文。从此Agent告别金鱼脑。

第五个，ARIS，1万2千星。全名叫Auto-Research-In-Sleep，翻译过来就是：睡觉的时候自动做研究。给它一个课题，它会自己读论文、跑实验、写报告，你醒来看结果就行。科研人的终极梦想。

第六个，enableMacosAI，1200星，6月10号发布。国行Mac用户的福音，一键开启完整Apple智能功能，包括端侧AI和Private Cloud Compute。以前折腾半天的配置，现在一行命令搞定。

第七个，GordenSuperPPTSkills，919星。号称AI PPT赛道终结者。用GPT生成豪华图片格式PPT，然后转成完全可编辑的PPTX。效果吊打市面上所有AI做PPT的工具。

以上就是六月GitHub AI月榜。关注我，每月追最前沿的AI技术动态。

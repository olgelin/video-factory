## 视觉模式使用规则

| 模式 | 使用场景 | 示例 |
|------|----------|------|
| **hero** | 开场/结尾/重要结论 | "今天给大家分享..." |
| **dashboard** | 数据指标/排行榜/多指标对比 | "销售额增长20%，用户数突破10万" |
| **compare** | 有明确两方对比数据 | "A方案 vs B方案"、"30人里20个选择..." |
| **flow** | 步骤/流程/时间线 | "第一步...第二步...第三步..." |
| **quote** | 引用/金句/名人名言 | "正如马云所说..." |
| **hud** | 未来感/科技感/预测 | "2025年趋势预测" |

## 配音注意事项

### 数字处理
- 2000 → 两千
- 10000 → 一万
- 50% → 百分之五十
- 3.5% → 百分之三点五

### 标点处理
- 句号用逗号替代（避免生硬停顿）
- 省略号保留（增加语气）

- 字幕: `narration/captions.srt`
- 时间戳: `narration/transcript.json`
